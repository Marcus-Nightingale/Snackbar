Snackbar.js

Snackbar.js is a lightweight package used to show notifications.

Example

Clicking any button will create a snackbar in the location which the button was pressed.

Example Usage
$.Snackbar(
    /*  Title  */ 'Hello World!',
    /* Message */ 'This is an example snackbar.',
    /*  Type   */ '',
    /* Options */
    {
        position: "snack-top-right",
        stack: false,
        timeout: 6000
    }
);
The above will result in this

Hello World!
This is an example snackbar.

Available Options
appendTo
String

Default: body

Sets the element for the snackbar wrapper to append to.

stack
Bool

Default: true

Allows the snackbars to stack. If set to false this will result into snackbars siting on top of each other.

position
String

Default: snack-bottom-right

Sets the location for the snackbar to be shown available options are;

  • snack-bottom-right
  • snack-bottom-left
  • snack-bottom-center
  • snack-top-right
  • snack-top-left
  • snack-top-center

timeout
Int

Default: 4000

The amount of time in milliseconds before the snackbar disappears.

progress
Bool

Default: true

Set if the progress bar at the bottom of the snackbar is shown.

closeable
Bool

Default: true

Allows the snackbar to be closed using the close icon.

icon
Bool

Default: true

Set if the snackbar icon should be shown.

rounded
Bool

Default: true

Changes if the snackbar has rounded corners or not.
